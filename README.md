# Access control
This project includes a client and a server in Java.

## Build
Build the jar files:
```shell
mvn package
```

It should have build `target/server.jar` and `target/client.jar`. 

## Run
You can then run it
```shell
java -jar target/server.jar
```
And
```shell
java -jar target/client.jar
```
